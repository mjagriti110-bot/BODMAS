# 🎮 BODMAS Interactive Game

An engaging and educational game designed to help learners understand and apply **BODMAS (Order of Operations)** through visual and interactive gameplay. Inspired by puzzle mechanics similar to traffic or sorting games (like Bus Jam), this project transforms math learning into a fun challenge.

---

## 🧠 Concept

The game teaches the correct order of operations:

**BODMAS = Brackets → Orders → Division → Multiplication → Addition → Subtraction**

Players solve math expressions by visually selecting and resolving operations in the correct sequence. Instead of solving traditionally, users interact with elements on the screen, making learning intuitive and memorable.

---

## 🎯 Features

* 🎲 **Interactive Gameplay**

  * Tap/click on parts of an expression to resolve them step-by-step.
* 🎨 **Visual Learning**

  * Expressions are broken into blocks, colors, or moving elements to represent priority.
* 🧩 **Puzzle-Based Levels**

  * Increasing difficulty with more complex expressions.
* ⏱️ **Timed Challenges**

  * Solve expressions within time limits for higher scores.
* 🧠 **Hint System**

  * Guides players when they make incorrect moves.
* 📊 **Progress Tracking**

  * Track performance, accuracy, and improvement.

---

## 🕹️ How to Play

1. A mathematical expression appears on the screen.
2. Each part of the expression is visually separated (like tiles or blocks).
3. Select the correct operation based on BODMAS rules.
4. The game resolves that step and updates the expression.
5. Continue until the final answer is reached.

✔ Correct choices earn points
❌ Wrong choices may reduce score or give feedback

---

## 📚 Example

Expression:
`8 + 4 × (6 - 2)`

Gameplay Flow:

1. Solve **(6 - 2)** → becomes `8 + 4 × 4`
2. Solve **4 × 4** → becomes `8 + 16`
3. Solve **8 + 16** → Final Answer: `24`
